# Skin Lesion Classification using Convolutional Neural Networks (CNN)

Deep learning project for automated classification of skin lesions using Convolutional Neural Networks (CNN) trained on the HAM10000 dermatology dataset.

This project explores the application of computer vision techniques in medical image classification to support early detection of skin cancer and assist clinical decision-making.

## Dataset

Dataset used: **HAM10000 ("Human Against Machine with 10000 training images")**

The dataset contains labelled dermatoscopic images of common pigmented skin lesions.

Dataset source:
https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/DBW86T
